# test_ai

## Overview
This project contains a Python script (`read_xml.py`) that reads and prints the structure of an XML file.

## Usage
1. Run the script using Python.
2. Provide the path to an XML file when prompted.

## Example
```
python read_xml.py
Enter the path to the XML file: example.xml
```

The script will output the tags, attributes, and text of the XML elements.
